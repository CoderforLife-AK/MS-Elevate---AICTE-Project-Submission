Radium Store Sales Analysis Dashboard

📌 Project Overview
This project features an interactive Power BI Dashboard developed to analyze the sales performance of Radium Store, an e-commerce entity operating across India. The goal was to transform raw, fragmented transaction data into actionable insights, helping the business identify growth opportunities, regional trends, and customer preferences.

🚀 Key Features
Real-Time KPI Tracking: Instant visibility into Total Profit, Revenue, Order Quantity, and Average Order Value.

Geographical Analysis: A dynamic map and chart system to identify top-performing states (e.g., Maharashtra, Uttar Pradesh) and specific cities.

Product Performance: Detailed breakdown of sales by Category (Clothing, Electronics, Furniture) and sub-categories.

Customer Insights: Analysis of top-spending customers and preferred Payment Methods (COD, UPI, Debit Cards).

Interactive Filters: Integrated Slicers for Quarters and States to allow for granular data exploration.

🛠️ Technology Stack
Business Intelligence: Microsoft Power BI Desktop

Data Transformation: Power Query (ETL)

Analytical Calculations: DAX (Data Analysis Expressions)

Data Source: CSV / Excel

📂 Project Structure
Data/: Contains the raw Madhav Store dataset (Orders and Details).

Dashboard/: The .pbix file containing the final Power BI report.

Screenshots/: Visual previews of the dashboard layers.

📈 System Development Approach
ETL Process: Cleaned and formatted raw data using Power Query to handle missing values and standardize naming conventions.

Data Modeling: Created a relational schema between transaction tables to ensure accurate cross-filtering.

Visualization: Designed a user-centric UI focused on high-level business metrics followed by detailed categorical breakdowns.

🔮 Future Scope
Predictive Analytics: Integrating Machine Learning to forecast seasonal sales trends.

Live Connectivity: Connecting the dashboard to a live SQL database for automated updates.

Advanced Segmentation: Implementing RFM analysis to categorize customer loyalty.
